ObjektOrientierung: Klassen, Vererbung, Polymorphie -> später AspektOrientierung: Erweiterung der ObjektOrientierung funktionale Programmierung: Funktionen können nicht nur angewendet werden, sondern als Daten genutzt usw...
  • interpretiert
  • dynamische Typisierung - Überprüfung von Datentypen zur Laufzeit
    • C: int a = 12;
    Python: a = 12
Interpretiert, nicht kompiliert -> Um ein Programm auszuführen braucht man einen Interpreter (Kompiliert: einmal kompilieren, immer ausführbar) relativ langsam: Interpretiert, speicherverwaltung, dynamisch >>Tafel: bei einer Zuweisung von x=2 und y=2 wert 2 wird angelegt; die variablen x und y haben eine Referenz auf den Wert 2 im speicher.
